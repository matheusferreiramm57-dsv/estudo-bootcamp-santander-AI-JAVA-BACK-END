# ☕ Java | Exercício - Cálculo de Idade

## 📖 Sobre

Este exercício foi desenvolvido durante meus estudos em Java no Bootcamp Santander Open Academy.

O objetivo foi praticar conceitos fundamentais da linguagem, como entrada de dados utilizando a classe `Scanner`, manipulação de variáveis, operações aritméticas e exibição de informações formatadas no console.

Criar um programa que:

- Solicite o nome do usuário;
- Solicite o ano de nascimento;
